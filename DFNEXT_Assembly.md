## Enclosure Assembly

1.  Install 4x bolts in back of B&R Forte FR303015 enclosure. M8x40 Bolt
    🡢 SS Washer M8 🡢 Nylon Washer M8 🡢 Enclosure 🡢 Nylon Washer M8 🡢 SS
    Washer M8 🡢 Nylock Nut M8 🡢 2x M8 SS Nut 🡢 Bolt rubber end cap
2.  Install grounding strap including star washers on either side of
    connector.
3.  Install 4x lock nuts M8 on the enclosure studs.

## Backplate Assembly

1.  Place EXT backplate on bench and install Commell LE-37GXIT7 computer
    using M3x12.0 bolt.
2.  Install mSATA OS SSD and dual-port SATA controller card in mPCI-E
    slots
3.  Pull black end of Serial Cable through bolt clearance hole under PCB
    with black connector facing upwards.
4.  Install blower fan with 2x M4x25.0mm bolt
5.  Install 2x VG Card guides with 2x M3x8.0mm bolt

## Computer Configuration

For proper operation, a few settings must be changed on the DFN
computer.

For pins marked set Jumpers to position 1-2 to enable AT Mode.

For the second set of pins marked, set jumpers to position 1-2 to enable
mSATA support. This setting ensures the PC detects the SSD storage.
